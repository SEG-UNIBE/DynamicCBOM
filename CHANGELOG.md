## [0.1.0] - 2025-01-06

### Added
- Initial release of DynamicCBOM
- eBPF-based runtime cryptography function tracing
- Support for OpenSSL 3.x tracing via bpftrace
- CycloneDX 1.6 CBOM format output
- Multiple tracing modes (attach to PID, global trace, run new target)
- Comparison metrics (precision, recall, F1-score)
- Publication-quality visualizations and charts
- Comprehensive documentation
